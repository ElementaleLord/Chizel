# Chizel - The Git We Need
Chizel acts like a GitHub clone but better, made by 4 Computer Science Students for their final project before their license, 
this project aims to push our abilities and minds to the absolute limit, while also having a high-risk-high-reward product if it makes it to the release.
Free from Microsoft's decisions, AI plantation, and spaghetti code, Chizel takes most if not all of GitHub's features whilst also trying
to make them better, faster and more efficient.

All coded in C, with the surveillance of a supervisor and advice from professors, we hope to achieve something big with this project
that we're pouring our hearts out on.
